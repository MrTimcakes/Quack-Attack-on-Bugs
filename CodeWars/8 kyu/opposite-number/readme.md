# Opposite number

URL: https://www.codewars.com/kata/56dec885c54a926dcd001095

Completed At: 2023-11-15T20:53:10.033Z

Difficulty: 8 kyu

Description:

Very simple, given an integer or a floating-point number, find its opposite.

Examples:
```
1: -1
14: -14
-34: 34
```
